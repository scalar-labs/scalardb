# ScalarDB Supported Databases

ScalarDB supports the following databases and the databases that are compatible with those databases.
* Cassandra
* Cosmos DB
* DynamoDB
* MySQL
* PostgreSQL
* Oracle Database
* SQL Server
* AWS Aurora MySQL
* AWS Aurora PostgreSQL
